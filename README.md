## 🚀 Project Overview

The Language Translator Tool is a web-based application that enables users to:

- Enter text in one language
- Select a target language
- Instantly receive AI-powered translations

The application integrates with the Google Gemini API to generate accurate and intelligent translations in real time.

---

## ✨ Features

- 🌐 Translate between multiple languages
- ⚡ Fast AI-generated translations
- 🎯 Simple and user-friendly interface
- 🔐 Secure API key handling using environment variables

## ⚙️ Installation & Setup

### 1️⃣ Prerequisites

Make sure you have the following installed:

- Node.js
- npm

Check versions:


node -v
npm -v


---

### 2️⃣ Install Dependencies

Open terminal inside the project folder and run:


npm install


---

### 3️⃣ Configure Environment Variables

Create a file named:


.env


Add your Gemini API key:


GEMINI_API_KEY=your_api_key_here


⚠️ Important:
- Do NOT upload your `.env` file to GitHub.
- Keep your API key private.
- Make sure `.env` is included in `.gitignore`.

---

### 4️⃣ Run the Application


npm run dev


The application will start on:


http://localhost:3000


Open this link in your browser to use the app.
